# DK Gaming Zone Image Map

Website-facing images live in these folders:

- `slider/`: homepage hero slider images
- `products/product1` to `products/product5`: product gallery images used by `js/data.json`
- `categories/`: category tiles for home and shop pages
- `campaigns/`: promotional banner cards
- `blogs/`: blog card and article images
- `brands/`: partner/tech badge icons
- `avatars/`: review/customer avatars
- `footer/`: compact footer badges and payment/store images

Original generated source images are stored in `generated/source/`.

When replacing product images, keep the existing paths in `js/data.json` unless you also update the JSON.
